How to Join Our Servers

×
1

Click Join on Server

Click join on the server you want on this page.

2

Enable Custom Files

In your options, make sure you have "Allow all custom server files" to ensure you download the content.

3

Alternative Method: Through Steam

1. Own Garry's mod on Steam
2. Navigate to the Multiplayer tab
3. Navigate to the Homigrad category
4. Enable "Allow all custom server files" in options
5. Join any server of your choice
