# zmod.gg

Official website for ZMOD

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- Vite - Build tool & dev server
- Vanilla JS (ES6 modules)
- CSS3
- GSAP for animations

## Project Structure

- `/src` - Entry points for each page
- `/js` - JavaScript modules
- `/css` - Stylesheets
- `/public` - Static assets
- `/dist` - Built output (generated)
